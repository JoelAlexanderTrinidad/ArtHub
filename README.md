### Información General
***
App web homebanking para gestionar cuentas bancarias de manera online. Creado con:
  * Backend
    - Java
    - Spring boot
    - JPA | Hibernate
    - Spring security
  * Frontend
    - HTML | Css | js
    - Vue js

  Instalaciones necesarias para correr la app web:
   - Intellij idea
   - JDK 11
   - Gradle
   - Clonar el proyecto.
   - Cambiar de rama a: git checkout local-server
